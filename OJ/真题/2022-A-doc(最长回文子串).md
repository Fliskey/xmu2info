# 最长回文子串

## 题目背景

本题改自经典算法题

## 题目描述

给你一个字符串 $s$ ，找到 $s$ 中最长的回文子串。如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。

## 输入格式

输入字符串 $s$ ，保证 $s$ 只包含小写字母，且 $1\le s.length \le3000$。

## 输出格式

输出字符串 $s$ 的最长回文子串。

## 样例 #1

### 样例输入 #1

```
babad
```

### 样例输出 #1

```
bab
```

## 样例 #2

### 样例输入 #2

```
cbbd
```

### 样例输出 #2

```
bb
```

## 提示

如果有多个长度相同的最长回文子串则只输出第一个。
