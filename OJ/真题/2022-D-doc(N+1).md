# N+1

## 题目描述

给定一个数，给出这个数$+1$后的结果

## 输入格式

输入一个数字$n$

## 输出格式

输出$n+1$后的结果

## 样例 #1

### 样例输入 #1

```
3
```

### 样例输出 #1

```
4
```

## 样例 #2

### 样例输入 #2

```
121
```

### 样例输出 #2

```
122
```

## 提示

| 数据点    | 数据范围                     |
| --------- | ---------------------------- |
| $1\sim2$  | $0 \le n \le 10^{10}$        |
| $3\sim10$ | $10^{10} \le n \le 10^{200}$ |
